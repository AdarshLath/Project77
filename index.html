Project On Console



Visual Studio Code is an IDE software. Following steps should be followed to set it up:
1. Download Visual Studio Code using this link
2. Create a new folder on your computer.
3. Open Visual Studio IDE and click on File > Open Folder, then browse and open the folder created in step 2.
4. Install Live-Server Extension if not already installed.



To find the values on the console, follow the mentioned steps:
1. Right click on your mouse.
2. Click "Inspect".
3. You will now see the values in the console.
